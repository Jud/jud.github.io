Operator

Talk to your terminal.

Push a button, say what you want — it routes to the right terminal agent and talks back. Fully on-device.

[ 30-second demo video ]

What it is

Voice orchestration for terminal agents. Run several Claude Code (or aider, codex, …) sessions at once and just talk — Operator transcribes, figures out which agent you mean, delivers the message, and speaks the reply back in that agent's own voice.

Why it's different

  • Multi-session routing — say "tell sudo to check the mempool" and it lands in the right session.
  • Voice in and out — each agent gets its own voice, so you know who's talking.
  • Fully on-device — WhisperKit for ears, Kokoro for mouth. No cloud, no API costs, no network round-trip.
  • Terminal-native — works with iTerm2 and Ghostty, not a browser tab.
